What's new in version 2.0 of Octave

Version 2.0.10 of Octave was released February 6, 1998. Most bugs reported since version 2.0 was release have been fixed, and a number of new features have been added. Octave is now much more compatible with MATLAB.

A list of user-visible changes in recent versions of Octave may be found in the file NEWS, distributed in both source and binary releases of Octave.
